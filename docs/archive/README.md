# Documentation Archive

but are not current product or implementation authority.

Use current authority docs before acting:

- `README.md`
- `AGENTS.md`
- `JOURNEY.md`
- `docs/DEVELOPMENT_PROCESS.md`

Archived files may describe older routes, schema assumptions, verification
commands, or product decisions that were superseded by the Pilot MVP baseline.
